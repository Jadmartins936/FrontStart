## Descrição
FrontStart é uma plataforma web educacional simples, inspirada em sites de ensino online, com foco no aprendizado de tecnologias front-end como HTML, CSS e JavaScript. O sistema organiza o conteúdo em módulos com teoria, exemplos práticos e exercícios, permitindo um aprendizado progressivo e intuitivo.

## 🚀 Funcionalidades

- Página inicial com apresentação do projeto  
- Tela de login simples (simulação de acesso)  
- Página de conteúdos (HTML, CSS e JavaScript)  
- Exibição de aulas com vídeos e exemplos práticos  
- Lista de exercícios interativos  
- Navegação entre módulos  
- Layout responsivo básico  

## ⚙️ Metodologia

O projeto é desenvolvido utilizando a metodologia ágil **Scrum**, com organização em **Sprints semanais**, garantindo entregas incrementais e maior flexibilidade.

- Organização de tarefas em ferramenta de gestão  
- Versionamento de código utilizando GitHub  
- Desenvolvimento contínuo e iterativo  

## 📦 Requisitos

- Navegador web atualizado (Google Chrome, Edge, Firefox, etc.)  
- Não requer instalação de dependências externas  

## 📁 Estrutura do Projeto

```bash
/plataforma-front-end
│
├── index.html
├── login.html
├── conteudos.html
│
├── /pages
├── /css
├── /js
├── /assets
└── README.md

```

## 🚀 Como executar

- Clone o repositório ou baixe o ZIP  
- Extraia os arquivos  
- Abra o arquivo `index.html` no navegador  

## 🚀 Uso

- Acesse a página inicial  
- Faça login (simulação)  
- Navegue pelos módulos (HTML, CSS e JavaScript)  
- Assista às aulas e acompanhe os exemplos  
- Pratique com os exercícios disponíveis
